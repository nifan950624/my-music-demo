# Vue 仿写网易云音乐

- 使用 **Vuex** 管理数据变化
- 使用 **axios** 调取接口
- 使用 **express** 实现代理请求
- 使用 **CSS3 动画** 实现播放动画

```
git clone https://github.com/nifan950624/my-music-demo.git
```


## 安装模块

```
npm i
```

## 启动项目

```
npm start  ||  npm run dev
```
